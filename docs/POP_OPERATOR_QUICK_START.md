## 📱 시작하기

### POP 앱 접속

#### 방법 1: 웹 브라우저
1. 태블릿 또는 스마트폰에서 Chrome/Safari 실행
2. 주소창에 입력: `https://your-company.mes.com/pop`
3. 로그인 (사번과 비밀번호 입력)

#### 방법 2: PWA 설치 (권장)
1. 위 주소 접속 후 "홈 화면에 추가" 클릭
2. 아이콘이 홈 화면에 추가됨
3. 앱처럼 실행 가능 (오프라인 지원)

### 로그인
1. **사번** 입력
2. **비밀번호** 입력
3. "로그인" 버튼 터치

## 📋 작업 시작하기

### 1단계: 작업지시 선택
1. "작업 지시" 메뉴 터치
2. 내 작업 목록에서 작업 선택
3. 또는 바코드 스캔으로 자동 선택

### 2단계: 작업 시작
1. 작업 카드에서 **"작업 시작"** 버튼 터치
2. 확인 메시지 확인
3. 타이머 자동 시작

### 3단계: 생산 수량 기록
1. 생산한 수량만큼 **+/- 버튼** 터치
2. 또는 숫자 직접 입력
3. **"기록"** 버튼 터치
4. 실시간 진행률 업데이트 확인

---

## ✅ 생산 수량 기록 방법

### 터치 입력 (권장)
```
┌─────────────────────────────────┐
│     생산 수량                    │
├─────────────────────────────────┤
│   [-]    [  50  ]    [+]        │
│                                 │
│  [+10] [+50] [+100]             │
└─────────────────────────────────┘
```

**사용법**:
- **[-]**: 수량 1개 감소
- **[+]**: 수량 1개 증가
- **[+10]**: 한 번에 10개 증가
- **[+50]**: 한 번에 50개 증가
- **[+100]**: 한 번에 100개 증가
- **숫자 직접 터치**: 키패드 입력

### 예시: 50개 생산 기록
1. **[+10]** 버튼 5번 터치
2. 또는 **[+50]** 버튼 1번 터치
3. 또는 숫자 영역 터치 → `50` 입력
4. **"기록"** 버튼 터치 → 완료!

## ❌ 불량 기록하기

### 불량 발생 시
1. **"불량 기록"** 버튼 터치
2. 불량 다이얼로그 표시

### 불량 정보 입력
```
┌─────────────────────────────────┐
│  불량 기록                       │
├─────────────────────────────────┤
│  불량 수량: [  5  ]              │
│                                 │
│  불량 유형: [외관 불량 ▼]        │
│                                 │
│  심각도:                        │
│   ○ 경미  ● 주요  ○ 심각        │
│                                 │
│  불량 사유:                     │
│  ┌─────────────────────────┐    │
│  │ 표면 스크래치 발견        │    │
│  └─────────────────────────┘    │
│                                 │
│  [사진 촬영 (선택)]             │
│                                 │
│  [취소]  [불량 기록]            │
└─────────────────────────────────┘
```

### 불량 유형 (7가지)
- **외관 불량**: 스크래치, 오염, 변색
- **치수 불량**: 크기, 두께, 길이 불일치
- **기능 불량**: 작동 안 됨
- **재질 불량**: 재질 결함
- **조립 불량**: 조립 오류
- **포장 불량**: 포장 손상
- **기타**: 기타 불량

### 심각도 선택
- **경미 (MINOR)**: 재작업 가능
- **주요 (MAJOR)**: 선별 필요
- **심각 (CRITICAL)**: 즉시 폐기

### 사진 촬영 (선택)
1. **"불량 사진 촬영"** 버튼 터치
2. 카메라로 불량 부위 촬영
3. 사진 자동 첨부

## 📄 SOP (표준작업절차) 따르기

### SOP란?
Standard Operating Procedure (표준작업절차)
- 작업 단계별 체크리스트
- 안전하고 정확한 작업을 위한 절차

### SOP 실행
1. 작업 시작 시 SOP 자동 표시
2. 또는 "SOP" 메뉴에서 확인

### 단계별 체크
```
┌─────────────────────────────────┐
│  [1] 원자재 준비 [필수] [중요]   │
│                                 │
│  원자재 재고 확인 및 준비        │
│                                 │
│  [통과 (Pass)]  [실패 (Fail)]   │
└─────────────────────────────────┘
```

**사용법**:
1. 단계 내용 읽기
2. 작업 수행
3. **통과** 또는 **실패** 선택
4. 실패 시 → 사유 입력 필수
5. 다음 단계 진행

### SOP 완료
- 모든 필수 단계 완료 필요
- 완료 버튼 활성화
- "SOP 완료" 터치

## 📱 바코드 스캔하기

### 스캔 가능 항목
- **작업지시**: 작업 자동 선택
- **자재**: 자재 입고 확인
- **제품**: 제품 식별
- **LOT**: 로트 추적

### 스캔 방법

#### 카메라 스캔 (권장)
1. "스캐너" 메뉴 터치
2. 카메라 자동 활성화
3. QR/바코드에 카메라 정렬
4. 자동 인식 (삐- 소리)
5. 결과 표시

#### 수동 입력
1. "수동 입력" 탭 선택
2. 바코드 번호 입력
3. "확인" 터치

### 스캔 팁
- 🔆 밝은 곳에서 스캔
- 📏 10-20cm 거리 유지
- 📐 바코드와 평행하게
- 🎯 흔들림 없이 고정

## 🌐 오프라인 모드

### 오프라인 모드란?
네트워크 연결 없어도 작업 가능!

### 작동 방식
```
┌─────────────────────────────────┐
│  ⚠️ 오프라인 모드               │
│  대기 중인 기록: 3건             │
│  [수동 동기화]                  │
└─────────────────────────────────┘
```

1. **오프라인 감지**: 자동 표시
2. **작업 계속**: 정상 작업 가능
3. **자동 저장**: 기록이 대기열에 저장
4. **온라인 복귀**: 자동 동기화 시작
5. **동기화 완료**: 모든 기록 서버 전송

### 오프라인 시 주의사항
- ⚠️ 작업은 가능하나 실시간 업데이트 안 됨
- ⚠️ 대기 중인 기록 확인
- ⚠️ 온라인 복귀 시 동기화 대기
- ✅ 동기화 완료 확인 필수

## ❓ 자주 묻는 질문 (FAQ)

### Q1: 로그인이 안 돼요
**A**: 다음을 확인하세요:
- ✅ 사번과 비밀번호 정확한지 확인
- ✅ 대소문자 구분 확인
- ✅ 네트워크 연결 확인
- ✅ 관리자에게 계정 활성화 확인

### Q2: 바코드 스캔이 안 돼요
**A**: 다음을 시도하세요:
- ✅ 카메라 권한 허용 확인
- ✅ 밝은 곳으로 이동
- ✅ 바코드 깨끗한지 확인
- ✅ 거리 조절 (10-20cm)
- ✅ 수동 입력으로 대체

### Q3: 오프라인 모드에서 작업이 저장되나요?
**A**: 네! 자동 저장됩니다.
- ✅ 모든 기록 대기열에 저장
- ✅ 온라인 복귀 시 자동 동기화
- ✅ 동기화 실패 시 재시도 (최대 3회)
- ✅ 실패한 기록 별도 표시

### Q4: 잘못 기록한 수량을 수정하려면?
**A**: 관리자에게 문의하세요.
- 운영자는 수정 불가
- 관리자가 수정 권한 보유
- 또는 역방향 기록으로 조정

### Q5: 작업 중 앱이 꺼지면?
**A**: 걱정 마세요!
- ✅ 작업 상태 자동 저장
- ✅ 다시 열면 이어서 진행
- ✅ 타이머 계속 동작
- ✅ 기록된 데이터 유지

### Q6: 불량 사진을 나중에 추가할 수 있나요?
**A**: 불량 기록 시에만 가능합니다.
- 기록 시 "사진 촬영" 선택
- 나중에 추가는 관리자에게 문의

### Q7: SOP 단계를 건너뛸 수 있나요?
**A**: 선택 단계만 가능합니다.
- 🔴 "필수" 표시 → 반드시 완료
- 🟡 "중요" 표시 → 권장
- ⚪ 일반 단계 → 선택 가능

### Q8: 여러 작업을 동시에 할 수 있나요?
**A**: 한 번에 하나만 가능합니다.
- 현재 작업 완료 또는 일시정지
- 다음 작업 시작
